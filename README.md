# Sputnik-launch-api
A lightweight Node.js API simulating telemetry and launch data for the historic Sputnik-1 mission. Built for demo and security scanning purposes.

## Features

- REST API built with Express
- Simulated launch status endpoint
- Telemetry data endpoint
- Dockerized for container deployment
- Designed for security scanning demo purposes

## Endpoints

- GET / → Basic welcome route
- GET /launch → Launch information
- GET /telemetry → Simulated telemetry data

## Purpose

This project is built as a demo application to test code, dependency, and container security scanning tools.
